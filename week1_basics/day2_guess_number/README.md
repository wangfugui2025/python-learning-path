\# Day2: 猜数字游戏

\## 功能描述

实现一个猜数字游戏，随机生成1-100的数字，用户猜测直至正确，并显示尝试次数

\## 使用方法

```bash

python main.py

```

\## 游戏规则

\- 程序随机生成1-100的整数

\- 玩家输入猜测的数字

\- 程序提示"偏大"或"偏小"

\- 猜中后显示总尝试次数

\## 技术实现

\- 使用random生成随机数

\- 异常处理确保输入为整数

\- 输入范围验证（1-100）

\- 尝试次数统计

\## 示例输出

```

欢迎来到猜数字游戏！

请输入1-100之间的数字：50

偏大

请输入1-100之间的数字：25

偏小

请输入1-100之间的数字：37

恭喜你猜中啦！共用了3次

```
